# AI Based Osteology Disease Detection and Clinical Suggestion System

## Project Description
This Final Year Project is based on Artificial Intelligence for Osteology Disease Detection and Clinical Suggestion System using Convolutional Neural Network (CNN). The system analyzes bone X-ray images to detect bone-related diseases and provides clinical suggestions based on prediction results.

## Technologies Used
- Python
- TensorFlow
- Keras
- CNN
- Google Colab
- NumPy
- Matplotlib
